# DIM – Dimension

DIM ist die zweite Station der BENCH-Pipeline.

Sie wird durch den RESPO-Träger **ORT** repräsentiert:

- RESPO.ORT → Stufe 1 → DIM

## Aufgabe
DIM stellt die Position, Lage und Einordnung dar.
Alle Werte sind transparent, aber systemintern artikuliert.

## Eigenschaften
- SYSTEM = OFF
- IKI = OFF
- MARKT = TRUE
- TRANSPARENT = TRUE
- INTERN = TRUE
